# Dotfiles
Dotfiles and current wallpapers for the setup on my Laptop

## What's been used?
- i3
- i3lock
- polybar
- rofi
- urxvt
- pywal
- compton

And I use Dotdrop to manage these dotfiles.

- run ./dotdrop.sh install
